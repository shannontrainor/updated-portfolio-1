GamerEvolution

GamerEvolution is the Web App my team created for Project 2. The purpose of this app is to provide gamers with a network to discover new game information, top game news, stats, scores, and connect with other gamers. The web app involves many features such as a search page where a user can search for any game and discover statistics about their chosen game, and even add chosen games to their "Gamer Library." There is also a profile feature where, upon Sign Up, a user can create their very own page, add videos, link Twitch or Youtube accounts, post status updates, chat with other gamers via the Discord widget, and add their own friends' profiles. There are also an extensive amount of animations throughout the app.

Employee Tracker

The Employee Tracker was developed for a homework assignment as a way for employers to record and update their employees, and view everything in a convenient table. This is utilized via the terminal, and npm install is required to install all packages prior to using the app. Once the app is running in the terminal, a list of choices is displayed for the user to pick from, including "View Employees", "View Department", "Add Employees", "Add Roles", and "Add Department", along with an "Exit" choice. The user is able to choose any option, which then triggers an inquirer.prompt method, sending the next question to the terminal. All data for this app is stored in a database using MySQL.

Project 1 - Image Gallery

For Project 1, my group and I developed an Image Gallery App. This app provides the user with a location to search for any type of image or GIF they would like. When the user inputs a keyword into the search bar and clicks the search icon, the image gallery will bring up images and GIFs related to the keyword. On the left side the user will find a dropdown with suggested Genres which will bring up images and GIFs releated to the genre when it is clicked. In addition, when an image or GIF is clicked on, a modal is dispayed on the screen where the user can "like" the image. This app was developed using Javascript, JQuery, ajax calls, and multiple API keys.
